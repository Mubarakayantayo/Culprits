## LLM-Integrated Honeypot for Intelligent Threat Deception

This project implements an AI-powered honeypot using the Beelzebub framework with LLM integration to simulate a real Linux system and deceive attackers.

The system captures attacker behavior, commands, and exploit attempts and visualizes activity using the ELK Stack.
